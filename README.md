# ArticleCrawling
뉴스 사이트 중 10개를 골라 원하는 카테고리를 입력해 그거에 맞는 기사를 크롤링해가지고 옵니다

뉴스 사이트의 종류는

중앙일보
머니투데이
내일 뉴스
YTN
국민일보
경향신문
해럴드뉴스
한겨레
동아일보
아시아머니로 총 10개입니다

**이 레포지스토리는 K-news-keyword-summarizer 레포지스토리로 merge되었습니다. 이후의 업데이트나 커밋은 전부 K-news-keyword-summarizer로 했기 때문에 거기서 확인해주시면 될것 같습니다**
